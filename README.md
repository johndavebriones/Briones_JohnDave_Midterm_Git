**What is GitHub used for?**
GitHub is a cloud-based platform for hosting Git repositories, enabling collaboration on software projects. It provides tools for version control, issue tracking, code review, and project management.

**What is the difference between Git and GitHub?**
Git is a distributed version control system that tracks changes in files locally on your machine. GitHub is a web-based service that hosts Git repositories online and adds collaboration features like pull requests and issues.

**Why is Branching useful?**
Branching allows you to create isolated environments to work on features or fixes without affecting the main codebase. It enables parallel development and makes it easy to experiment and merge changes when ready.

**What command did you use to upload your commits to GitHub?**
The command `git push` is used to upload local commits to a remote repository on GitHub. For a new branch, `git push -u origin <branch-name>` sets the upstream tracking.
